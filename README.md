# AI Phishing Website Detector

## Overview

AI Phishing Website Detector is a Machine Learning based web application that identifies whether a website URL is legitimate or phishing.

## Features

- URL Analysis
- Phishing Detection
- Machine Learning Prediction
- Flask Web Interface

## Technologies Used

- Python
- Flask
- Scikit-Learn
- Pandas
- HTML
- CSS
